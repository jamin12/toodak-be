## 메모
- 한 단계 끝날 때마다 체크리스트 업데이트 + 핵심 결정 변경 시 맥락노트.md 동기화
- Phase 3에서 Member의 `withdrawnAt`은 Entity·DDL에는 두되 UseCase에선 사용 안 함 (탈퇴 기능 추후)
- Phase 3에서 `LoginWithGoogleUseCase`의 RefreshToken 반환은 일단 빈 placeholder — Phase 4에서 실제 값으로 채움
- Google ID Token의 `audience(client_id)`는 환경변수로 외부 주입 (`GOOGLE_AUDIENCE` 등)
- JWT 시크릿도 환경변수 외부 주입 (`JWT_SECRET`)
- 로컬 개발용 PostgreSQL은 docker-compose 또는 testcontainers로 띄움
